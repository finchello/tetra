# Code of Conduct

## Our pledge

We want tetra to be a welcoming, harassment-free project for everyone,
regardless of background or experience level.

## Expected behaviour

- Be respectful, considerate, and constructive.
- Welcome newcomers and help them get started.
- Accept feedback gracefully and give it kindly.
- Focus on what is best for the project and its community.

## Unacceptable behaviour

- Harassment, discrimination, or personal attacks.
- Trolling, insults, or derogatory comments.
- Publishing others' private information without consent.
- Any conduct that would reasonably be considered inappropriate in a
  professional setting.

## Enforcement

Maintainers may remove, edit, or reject contributions, comments, and other
interactions that violate this Code of Conduct, and may ban repeat offenders.

## Reporting

Report concerns by opening a confidential issue or contacting the maintainer
through their GitHub profile.
